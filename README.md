# Quiz HTML, CSS and JavaScript

## Español

Este proyecto es un juego tipo quiz creado con **HTML**, **CSS** y **JavaScript**.

El objetivo del juego es responder preguntas básicas sobre programación web, especialmente sobre HTML, CSS y JavaScript.

## Funcionalidades

* Pantalla de inicio
* 10 preguntas
* Respuestas correctas e incorrectas
* Mensajes de acierto y error
* Sistema de puntuación
* Pantalla final con el resultado
* Diseño responsive
* Fondo visual relacionado con programación

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript

## Estructura del proyecto

```text
proyecto-quiz/
│
├── index.html
├── css/
│   └── generales.css
├── scripts/
│   └── funciones.js
└── img/
    └── imágenes del proyecto
```

## Cómo usarlo

1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en el navegador.
3. Pulsar el botón **Empezar**.
4. Responder las preguntas del quiz.
